## 截图建议

1. 浏览器窗口宽度尽量保持一致，推荐使用 1366×768 或 1440×900。
2. 截图前先导入演示数据，保证页面内容充足。
3. 首页截图建议使用管理员、员工、家属三个账号分别截取。
4. 业务模块截图建议至少保留一张列表页和一张新增或编辑弹窗。
5. 论文中不需要放所有截图，可以优先选择登录页、首页、核心业务模块和权限对比页面。
