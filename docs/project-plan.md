# 中小型养老院信息管理系统项目说明

本项目是毕业设计《中小型养老院信息管理系统的设计与实现》的基础工程。

## 技术栈

后端：

- Java 17
- Spring Boot
- MyBatis-Plus
- MySQL
- Maven

前端：

- Vue 3
- Vite
- Element Plus
- Axios
- Vue Router
- Pinia

## 当前阶段

当前只完成基础项目结构：

- 后端 Spring Boot 启动类和基础配置
- 前端 Vue 3 + Vite 基础入口
- 数据库建表 SQL 初稿
- 项目说明文档

后续将按模块逐步开发登录、老人信息、员工信息、房间床位、健康档案、护理记录、缴费和公告管理。
