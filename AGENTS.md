\## 项目信息



项目名称：《中小型养老院信息管理系统的设计与实现》



这是一个毕业设计项目，目标是做出可以运行、可以答辩演示、结构清晰的前后端分离管理系统。



\## 技术栈



后端：

\- Java 17

\- Spring Boot

\- MyBatis-Plus

\- MySQL

\- Maven



前端：

\- Vue 3

\- Vite

\- Element Plus

\- Axios

\- Vue Router

\- Pinia



\## 核心模块



系统包含：

1\. 登录功能

2\. 老人信息管理

3\. 员工信息管理

4\. 房间床位管理

5\. 健康档案管理

6\. 护理记录管理

7\. 缴费管理

8\. 公告管理



\## 工作规则



1\. 不要一次性生成整个系统。

2\. 每次只完成一个模块。

3\. 修改前先列出准备创建或修改的文件。

4\. 不要随便删除已有代码。

5\. 不要重构无关代码。

6\. 优先保证项目能运行、能演示、结构清晰。

7\. 不要读取 node\_modules、target、dist、.git、logs 等无关目录。

8\. 遇到报错时，只分析相关文件和相关日志。

9\. 代码不要过度复杂，适合毕业设计展示即可。
